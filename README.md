# Sistema de Orientación Médica

Aplicación desarrollada en Java que tiene como objetivo orientar a los pacientes sobre el tipo de atención médica que requieren según sus síntomas, edad, antecedentes y condición de salud.

El sistema permite registrar consultas, identificar grupos de riesgo y entregar recomendaciones sobre el establecimiento de salud al que deberían acudir, facilitando el acceso oportuno a la atención médica.
